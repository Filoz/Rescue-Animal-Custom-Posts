A WordPress plugin for themes developed by Rescue Themes

=== Description ===

When activated, this plugin will create a custom post type and add a menu item called "Animals" to the WordPress admin area. The custom posts have the ability to display a slideshow of an animal and display the animal's species, breed, gender, age, rehoming status, and special needs.

=== Installation ===

1. Upload the plugin to `/wp-content/plugins/`. Alternately, you can download this package as a zip and upload it via the WordPress dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the Animals menu to begin adding animals.
